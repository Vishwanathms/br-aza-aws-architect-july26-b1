Home Assignments 03

Focus: AWS Containers, Auto Scaling, Application Load Balancer, Amazon ECS, Amazon S3, Amazon RDS, Backup & Recovery, Cloud Architecture

Task: Modernize the Employee Management Platform by designing and deploying a highly available, secure, and scalable AWS solution using containerized applications, S3 storage, RDS databases, Auto Scaling, and disaster recovery capabilities.

Requirements:
•Containerize and deploy the Employee Portal using Docker.
•Configure Auto Scaling and Multi-AZ deployment for high availability.
•Deploy and test an Application Load Balancer.
•Evaluate and deploy the application using Amazon ECS.
•Create a secure S3 bucket with versioning and encryption.
•Implement lifecycle policies for storage cost optimization.
•Deploy an Amazon RDS MySQL database in a private subnet.
•Configure database security and connectivity.
•Create and validate backup and recovery procedures.
•Document architecture, security controls, and deployment decisions.

Expected Outputs:
•Employee Portal successfully deployed as a containerized application.
•High availability achieved using Auto Scaling and Load Balancing.
•ECS deployment implemented and evaluated against EC2-based Docker deployment.
•Secure document storage configured in Amazon S3.
•Automated storage optimization through lifecycle policies.
•RDS database deployed securely with backup capabilities.
•Backup and restore procedures validated successfully.
•AWS security and scalability best practices implemented throughout the solution.
