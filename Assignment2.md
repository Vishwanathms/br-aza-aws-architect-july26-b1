Focus:AWS Networking, VPC Design, Multi-Tier Architecture, Load Balancing, Route 53, Network Security

Task:Design and deploy a secure multi-tier application network on AWS using VPC, public and private subnets, NAT Gateway, Bastion Host, Application Load Balancer, Route 53, Security Groups, and NACLs.

Requirements:
•Create a production VPC with public, private application, and private database subnets across two Availability Zones.
•Configure Internet Gateway and NAT Gateway for secure internet connectivity.
•Deploy a Bastion Host and private application servers.
•Implement Security Groups and Network ACLs to control network access.
•Configure an Application Load Balancer with health checks and target groups.
•Set up Route 53 DNS to route traffic to the application.
•Perform network connectivity and security validation.

Expected Outputs:
•Secure and scalable AWS network architecture deployed.
•Public and private subnet communication configured correctly.
•Bastion Host providing secure administrative access.
•Application Load Balancer distributing traffic across servers.
•Route 53 DNS resolving application traffic successfully.
•Security controls implemented using Security Groups and NACLs.
•Network architecture validated according to AWS best practices.
