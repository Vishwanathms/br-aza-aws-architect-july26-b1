On **Amazon Linux 2023 (Amazon Linux 3)**, the recommended way to install the MySQL command-line client is to install the **MariaDB client**, which provides the `mysql` command and is fully compatible with MySQL servers. ([Amazon Web Services Documentation][1])

### Install MySQL Client (Recommended)

```bash
# Update packages
sudo dnf update -y

# Install MySQL/MariaDB client
sudo dnf install -y mariadb105

# Verify installation
mysql --version
```

Expected output:

```text
mysql  Ver 15.1 Distrib 10.5.x-MariaDB
```

### Connect to a MySQL Server

```bash
mysql -h <hostname> -P 3306 -u <username> -p
```

Example:

```bash
mysql -h mydb.abcdefghij.us-east-1.rds.amazonaws.com -u admin -p
```

---

## If you specifically need the Oracle MySQL client

Amazon Linux 2023 does not include the Oracle MySQL client in its default repositories. You must add Oracle's MySQL repository and install it from there. ([Repost][2])

```bash
sudo dnf install -y https://dev.mysql.com/get/mysql80-community-release-el9-4.noarch.rpm

sudo dnf install -y mysql-community-client

mysql --version
```

---

### Check installed client

```bash
which mysql
mysql --version
```

If you're connecting to **Amazon RDS MySQL**, **Aurora MySQL**, or a standard MySQL server, the `mariadb105` client is usually the simplest and recommended choice. ([Amazon Web Services Documentation][1])
