Focus:AWS IAM, Amazon S3, AWS Budgets, AWS Secrets Manager, Cloud Security

Task:Configure a secure AWS environment for an employee portal by implementing user management, secure storage, cost monitoring, secrets management, and security validation.

Requirements:
•Create a monthly budget alert with email notifications.
•Create IAM users and groups and assign users appropriately.
•Create an S3 bucket and configure read-only access for developers.
•Enable Block Public Access and Versioning on the bucket.
•Store database credentials securely in AWS Secrets Manager.
•Validate permissions and perform a security audit.

Expected Outputs:
•Budget monitoring configured successfully.
•IAM users, groups, and permissions implemented.
•Secure and private S3 storage configured.
•Secrets stored securely using Secrets Manager.
•Access controls validated according to requirements.
•Security audit completed and documented.
